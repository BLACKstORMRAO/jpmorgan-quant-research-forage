📌 Program Overview
This virtual experience simulated the work done by JPMorgan Chase's Quantitative Research (QR) team — one of the most prestigious quant teams in global finance. The program involved pricing financial contracts, building predictive models, and working with real-world financial datasets.

🗂️ Tasks Completed
Task 1 — Investigate and Analyse Price Data

Analysed historical natural gas price data
Identified seasonal trends and price patterns
Extrapolated prices to estimate future values
Built a price estimation model using interpolation techniques

Task 2 — Price a Commodity Storage Contract

Built a prototype pricing model for a natural gas storage contract
Inputs: injection/withdrawal dates, prices, storage costs, rate limits
Calculated the value of a contract given buy/sell strategy
Applied financial reasoning to optimise contract value

Task 3 — Credit Risk Analysis (PD Modelling)

Worked with a loan book dataset
Built a model to estimate Probability of Default (PD) for borrowers
Converted FICO scores into categorical buckets
Applied log-likelihood optimization to find optimal bucket boundaries

Task 4 — Mortgage-Backed Securities (FICO Bucketization)

Developed a dynamic programming solution for FICO score bucketization
Optimized boundaries to maximize predictive power
Converted continuous credit scores into meaningful risk categories
Applied quantitative methods used in real credit risk modelling


💡 Key Learnings

How quant researchers price financial derivatives
Commodity contract valuation methodology
Credit risk modelling and Probability of Default (PD)
Log-likelihood optimization for bucketing problems
Applying Python to real finance problems
